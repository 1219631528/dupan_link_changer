# 度盘秒传链接格式转换器
支持 pandownload/梦姬/PCS-Go/游侠/bdlink 链接格式

> * 单个游侠链接支持多行录入，需要连续按两次回车键才开始链接转换
> * 其他链接只支持单行录入，只需要按一次回车键就开始链接转换
> * PCS-Go格式在使用时候不要复制前缀BaiduPCS-Go，从rapidupload开始复制就行
> * 退出请直接关闭此窗口

支持python2.7+，建议使用python3+ （因为python2容易出现乱码问题）
